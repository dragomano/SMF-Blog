## Особенности

* Модификация добавляет только тег — `[jstex]` для вставки формул, без дополнительных кнопок.

Образец кода:

```bbcode
[jstex]\lim_{n \to \infty}\sum_{k=1}^n \frac{1}{k^2}= \frac{\pi^2}{6}[/jstex]
```
